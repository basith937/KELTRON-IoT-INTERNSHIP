# IoT Internship Program – KELTRON

## Overview

This repository contains the projects, experiments, and learning outcomes completed during the **Internet of Things (IoT) Internship Program at KELTRON (Kerala State Electronics Development Corporation Limited)**.

The internship provided hands-on experience in embedded systems, microcontroller programming, sensor interfacing, display technologies, wireless communication, and IoT application development using **Arduino Uno** and **ESP32** platforms.

The projects were designed to strengthen practical knowledge of IoT systems by integrating hardware and software components to develop real-world solutions for monitoring, control, and automation.

---

## About KELTRON

**KELTRON (Kerala State Electronics Development Corporation Limited)** is one of India's pioneering public sector electronics enterprises. Established in 1973, KELTRON has contributed significantly to the development of electronics, information technology, industrial automation, communication systems, and training programs across various engineering disciplines.

The IoT Internship Program offered practical exposure to modern embedded technologies and industry-oriented project development methodologies.

The primary objectives of the internship were:

# Projects Completed

## 1. OLED Display Interface Project

### Description

Implemented a graphical user interface on an SSD1306 OLED display using the Adafruit GFX and SSD1306 libraries.

### Features

* Text rendering
* Custom bitmap graphics
* Logo display
* User interface design

### Concepts Learned

* I2C communication
* OLED display programming
* Memory optimization using PROGMEM

---

## 2. Smart Parking Sensor System

### Description

Designed and implemented a parking assistance system using an HC-SR04 ultrasonic sensor, OLED display, and buzzer.

### Features

* Real-time distance measurement
* Distance display in centimeters and inches
* Audible warning alerts
* Dynamic buzzer response based on proximity

### Components

* Arduino Uno
* HC-SR04 Ultrasonic Sensor
* SSD1306 OLED Display
* Piezo Buzzer

### Applications

* Vehicle parking assistance
* Obstacle detection systems
* Distance monitoring solutions

---

## 3. ESP32 Web Server Motor Control

### Description

Developed a web-based motor control system using ESP32 and Wi-Fi connectivity.

### Features

* Embedded web server
* Wireless motor control
* Forward, reverse, and stop operations
* Browser-based interface

### Components

* ESP32 Development Board
* Motor Driver Module
* DC Motor

### Applications

* Home automation
* Robotics
* Remote control systems
